# Dev Environment Setup - Web Server

## Overview

This document describes how to set up the development environment for the web server.

The web server is built using C# and ASP.NET.

---

## Requirements

- .NET SDK (version 6 or higher)
- Visual Studio or JetBrains Rider
- Git

---

## Setup Steps

1. Clone the repository

git clone <repo-url>

2. Open the project in Rider or Visual Studio

3. Restore dependencies

dotnet restore

4. Run the project

dotnet run

---

## Result

The web server should start running locally.

You can access it at:

http://localhost:5000
