# Dev Environment Setup - API

## Overview

This document describes how to set up the API development environment.

The API is built using C# and ASP.NET Web API.

---

## Requirements

- .NET SDK
- Rider or Visual Studio
- Git

---

## Setup Steps

1. Clone the repository

git clone <repo-url>

2. Open API project

3. Restore dependencies

dotnet restore

4. Run API

dotnet run

---

## Result

The API server will run locally.

Example:

http://localhost:5000/api/health
